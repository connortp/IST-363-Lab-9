Corgi Videos

Start by populating the Corgi Videos section. An AJAX request will need to be send to YouTube's API and ask it for videos related to a Corgi query

  • After the key variable, use jQuery's AJAX method to create a new AJAX request
    • The base url should be set to https://www.googleapis.com/youtube/v3/search
    • Add a data property (that's an object) to pass in a set of parameters with the url
    • Add a key parameter and set it equal to the key variable
    • Add a maxResults parameter and set it equal to 8 (Make 8 a string, instead of a normal number)
    • Add a part parameter and set it equal to snippet. Snippet is a value used by YouTube's API to instruct it what type of information to send back. This can also include information about statistics, user information, etc.
    • Add a type parameter and set it equal to video
    • Add a q paramter and set it to Corgi. Q stands for query; what search term we want to use when getting videos back from YouTube's API
  • Chain a done method after the AJAX one. Pass through an annonymous function with a response parameter. This paramter can be named whatever you like. It will represent the response, or data, returned from YouTube.
  • Inside the done function, console out response to see what YouTube sent us back
