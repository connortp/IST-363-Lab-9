Search Feature

In addition to these three great section automatically loaded for visitors, they should also be able to search for their own preferred dog videos using the search box at the top of the site.

  • Use jQuery's keypress method to attach an event listener to the search input. This will fire whenever a user presses a key while typing inside the search box
  • The keypress method will take an annonymous function and then pass through an event parameter to that function
  • The code to fire off a request to YouTube and deal with other search functionality should only happen when the user presses the "Enter" key, not just any key. Inside the annonymous function, create an if conditional and test for if event.keyCode is equal to 13. Each key on a keyboard has a numeric code assigned to it. 13 represents the Enter key
  • Inside the conditional, prevent any default behavior from the browser (like submitting the form and having the page reload when we don't want it to), by preventing the default action on the event

Let's start working on the searching functionality now

  • Create a variable named query and set it equal to the value of the search input
  • Construct another AJAX request to YouTube's API. The type should be video, limit it to only 4 results, and set the q parameter equal to the concatenation of the query variable and t.he string " dog". Notice the space before the word dog
  • If the user searches for the term "Christmas", the search term sent to YouTube will be "Christmas Dog", ensuring DoggoTube remains all about those Doggos.
  • If the request was successful and data was returned, create the same <li> appended elements like the Corgi section on the unordered list inside the hidden search section on the page
  • After the append method, but still within the done function, set the value of the search box to be nothing (clear it out after the user searched)
  • Set the text value of the element with the id search-text equal to the query variable
  • Also use jQuery's show method to turn the .search-results section from display:none to display:block
