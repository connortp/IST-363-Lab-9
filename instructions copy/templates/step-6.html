Husky Playlists and Daschund Channels

Make similar AJAX calls and build out the list of data for the next two sections

Husky Playlists

For the Husky Playlists sections:

  • Set the q paramter to be Husky
  • Set the type parameter to be playlist
  • Append the markup to the unorder list with the class of playlist-list
  • Point the <a> href to https://youtube.com/playlist?list= and then the id of the playlist

Dachshund Channels

For the Dachshund Channel sections:

  • Set the q paramter to be Dachshund
  • Set the type parameter to be channel
  • Append the markup to the unorder list with the class of channel-list
  • Point the <a> href to https://youtube.com/channel/ and then the id of the channel
