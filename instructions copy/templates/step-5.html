Corgi Response

In the console of your browser, you should see something similar to below. YouTube returned data in a JSON format. Clicking on the arrows will expand the objects. YouTube's API return a list of 8 Corgi-related videos to your webpage (inside the code. Inside each video's snippet object is information about that video: it's title, description, published date, image thumbnails, etc.

Let's work with this data and use it to build out a list of these videos onto DoggoTube.

  • After the console.log statement, use jQuery's each method to loop through the list of videos. To access properties of the JSON object, dot notation is used. Iterate over response.items
  • The jQuery each method takes a second parameter that will be an annonymous function. Pass this function two paramters: index and video
  • Inside this function, append an <li> to the unordered list that has the class videos-list inside the Corgi section
    • Add a custom data-video-id attribute on the <li> and give it a value of video.id.videoId (This will need to be outside of a string and will access this specific property on the JSON object you're working with)
    • Inside the list item, create an <a> tag with an href which is the concatenation of "https://youtube.com/watch?v=" and the video id again
    • Inside the <a> tag, create an image whose src is equal to the url of the medium sized thumbnail for the video
    • After the image, but still within the link, create a <div> with the text Video: and then the title of the video
