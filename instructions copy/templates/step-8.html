Create A Video Modal

Instead of sending users to YouTube's site to watch videos, let's embed them in a modal on DoggoTube when a user clicks on one of them

  • Since the videos don't exist in the DOM when the document ready function fires, event listeners can't be attached directly to the <li>'s in each of the video-lists (AJAX requests happen Asynchronously)
  • Add a on method to $(document). Document is an element that represents the entire webpage. Set the event to be click, the selector to be the list items inside a video list, and the third parameter should be an annonymous function with an event parameter
  • Inside the function, prevent the default action browsers would perform to prevent sending users to YouTube's website should they click on the link
  • Create a variable called src and assign it a value of the data-video-id attribute of the element that triggered the click event
  • Create a variable called iframeSrc and assign it a value of the concatenation of https://youtube.com/embed/ and the src variable
  • Add the class visible onto the modal on the site
  • Set the inner HTML of the element that has the class modal-player to be an iframe. The iframe should have:
    • A height of 315
    • A width of 506
    • A frameborder of 0
    • Allow it to be fullscreen
    • Set the src of the iframe to be the iframeSrc variable you created
